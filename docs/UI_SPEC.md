# 猎影 UI 设计规范

## 配色方案

| 名称 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#0a1628` | 整体背景 |
| 侧边栏 | `#0d1b2a` | 左侧导航背景 |
| 内容区 | `#16213e` | 主内容区背景 |
| 强调色 | `#00d9ff` | Logo、按钮、高亮 |
| 辅助强调 | `#4facfe` | 渐变、hover |

## 组件样式

### 侧边栏
- 固定左侧，高度100vh
- Logo顶部居中，padding: 20px
- 导航项：图标 + 文字，左侧对齐
- 选中项：背景 #16213e，文字 #00d9ff
- 底部：用户头像(40x40) + 用户名

### 按钮
- 主按钮：渐变 #4facfe → #00f2fe，圆角 20px
- 次按钮：透明背景，边框 #2a3a5a，圆角 20px
- Hover：scale(1.05)，阴影

### 标签
- 圆角 20px，背景 rgba(255,255,255,0.05)
- 边框 1px #2a3a5a
- Hover：背景 rgba(79,172,254,0.2)

### 搜索框
- 高度 48px，圆角 24px
- 边框 2px #2a3a5a
- Focus：边框 #00d9ff
